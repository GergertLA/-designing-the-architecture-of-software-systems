# Определение тональности и степени влияния новостных событий на Bitcoin с применением методов машинного обучения

_Гергерт Людмила Андреевна
РИС-22-4_

Компания хочет создать аналитическую систему для прогнозирования волатильности Bitcoin на основе анализа новостного фона в реальном времени.

**Пользователи:** десятки трейдеров и аналитиков компании, потенциально — тысячи сторонних подписчиков через платную подписку.

**Требования:**

* система должна автоматически собирать новости из заданных источников (криптовалютные СМИ, соцсети, финансовые новости);
* для каждой новости необходимо определять тональность (позитивная, негативная, нейтральная) с указанием степени уверенности модели;
* необходимо определять тему новости (регуляторные новости, технологические обновления, макроэкономика, принятие институтами, хакерские атаки и т.д.);
* система должна коррелировать всплески определенной тональности и тематики с изменениями цены BTC, выявляя наиболее влиятельные типы событий;
* трейдеры должны получать автоматические оповещения о выходе "сильнонегативных" или "сильнопозитивных" новостей
* система должна предоставлять дашборд с визуализацией: динамика тональности новостей, рейтинг самых влиятельных источников, тегирование новостей по темам.

**Дополнительный контекст:**

* ручной анализ новостей не успевает за скоростью рынка, что приводит к финансовым потерям;
* конкуренция среди хедж-фондов в крипто-нише крайне высока, и любое аналитическое преимущество критически важно;
* новостной фон крайне изменчив, и модель должна постоянно дообучаться на новых данных, чтобы учитывать появление новых тем и изменение контекста.
